Case built around embedded projects based on the esp32 microcontroller and nrf2.4 radio modules 
including a .96'' OLED display (ssh1306) 4 button module. Designed to 
allow up to 6 nRF modules plus an additional Aux SMA connector out for an
external antenna for the esp32 when used with f_jammer firmware by worthlessoul
to allow remote triggering.
Configuration also creates compatability with with custom implementations of the 
nRFbox device design and firmware created by Cifertech.
Can easily be adapted/modified for many of the esp32/nRF projects
that commonly use these modules with the .96"OLED display.
